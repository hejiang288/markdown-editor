关于 Markdown 编辑器

我们的使命

为中文内容创作者提供一个简单、优雅的 Markdown 排版工具，特别优化微信公众号的内容排版，让技术写作变得更加轻松愉快。

无论您是技术博主、内容创作者、还是技术文档撰写者，我们的工具都能帮助您专注于内容创作，不必担心排版问题。

核心功能

  • 多主题支持：提供多种精美主题，满足不同场景需求
  • 代码高亮：对各种编程语言提供专业的代码高亮，自动适配主题风格
  • 样式保留：复制和导出时完整保留主题样式，确保一致的视觉体验
  • 微信优化：特别优化了微信公众号内容排版
  • 图片处理：智能调整图片大小，支持懒加载和错误处理，优化复制体验
  • HTML转换：粘贴HTML内容自动转为Markdown
  • 拖放支持：支持拖放Markdown文件到编辑区
  • 导出功能：一键下载HTML格式的文章

技术栈

我们的编辑器采用简洁高效的技术栈，保证了轻量级和高性能：

HTML5
CSS3
JavaScript
CodeMirror
Marked.js
Highlight.js
Turndown.js

编辑器完全在前端运行，无需后端服务器，您的内容不会通过网络传输，保证了数据的安全性和隐私。
